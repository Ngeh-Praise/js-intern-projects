# JS Intern Projects

This repository contains 3 small JavaScript projects completed as part of an internship exercise.

## Projects

### 1. Expense Tracker
- Add expenses with name and amount
- Delete or clear expenses
- Highlight expenses above 10,000
- Built with HTML, CSS, JavaScript

### 2. Student Grade UI
- Enter student name and score
- Check grade with letter A-F
- Color-coded results
- Built with HTML, CSS, JavaScript

### 3. Theme Switcher
- Switch between Light, Dark, and Blue mode
- Smooth transition and active button highlight
- Built with HTML, CSS, JavaScript
